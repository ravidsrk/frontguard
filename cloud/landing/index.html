How it works

Three steps. Sixty seconds.

Add the GitHub Action to your repo. We handle the rest.

1

Connect your repo

Add the Frontguard GitHub Action — three lines of YAML in your workflow file.

# .github/workflows/vrt.yml
- uses: frontguard/action@v1
  with:
    project: my-app
2

We screenshot everything

Cloud rendering captures every route across Chromium, Firefox, and WebKit in multiple viewports.

3

See what changed

AI-powered diff reports appear on every PR. Regressions, intentional changes, and content updates — classified automatically.
