# Frontguard Docs

Documentation site for Frontguard, built with [Fumadocs](https://fumadocs.vercel.app/) and Next.js.

## Development

```bash
npm run dev
```

Runs at `http://localhost:3000`.

## Build

```bash
npm run build
```

## Content

Documentation pages live in `content/docs/` as MDX files. The sidebar structure is auto-generated from the file tree.
