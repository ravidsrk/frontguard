Add visual testing to Playwright in 3 lines.

Add visual regression testing to your Playwright suite in under a minute.

Step 1 — Install

npm install @frontguard/playwright

One package. No config files, no dashboard setup.

Step 2 — Add to tests

import { test } from '@playwright/test';
import { visualTest } from '@frontguard/playwright';

test('checkout page', async ({ page }) => {
  await page.goto('https://myapp.com/checkout');
  const result = await visualTest(page, 'checkout');
  expect(result.passed).toBe(true);
});

Drop it into any existing Playwright test. Just 3 lines of code.

Step 3 — Run in CI

npx playwright test

Get PR comments with visual diffs and AI analysis.

Built for the problems pixel diffs can't solve.

AI-Powered Analysis — It tells you why, not where

Instead of "pixels differ at coordinates 340,890," you get: "the submit button overflows on mobile because the new padding pushes it outside the flex container." GPT-4o vision reads the screenshot the way a senior engineer would.

Anti-Flake Consensus — 3 screenshots, majority wins

Spinner mid-animation? Loading flicker? A blinking cursor in your hero? Frontguard takes three captures, keeps the consensus, drops the outlier. Flake noise stops drowning real regressions.

Smart Route Discovery — Point at a URL. It tests every page.

No config files. No route lists. No baseline directories to maintain. The crawler walks your sitemap, finds every reachable page, and builds the test matrix for you.

PR-Native Review — Diffs render in the GitHub thread

Every PR gets a comment with side-by-side visual diffs and the AI explanation inline. Approve or reject without leaving the review tab.

Multi-Browser — Chromium, Firefox, WebKit — one command

That Safari flexbox bug a customer found last quarter? It would have been caught at PR time. WebKit gets the same screenshot pipeline as Chromium.

Plugin System — 6 lifecycle hooks for whatever you need

Compare production against Figma. Set performance budgets. Monitor live pages. Build your own with onBeforeCapture, onAfterDiff, and onReport hooks.
