# 🛡️ Frontguard

AI-powered frontend visual regression testing. Detect, understand, and fix visual bugs before production.

## Quick Start

```bash
npx frontguard init
npx frontguard run --url http://localhost:3000
```

## Features

- Zero-config route discovery
- Multi-browser (Chromium, Firefox, WebKit)
- AI-powered regression analysis (BYOK)
- Smart rendering (dependency graph)
- Vercel/Netlify preview URL auto-detection
- Git orphan branch baseline storage
